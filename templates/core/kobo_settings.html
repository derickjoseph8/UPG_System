How to Get Your API Token
  1. Go to https://kf.kobotoolbox.org and log in
  2. Click on your username (top right) → Account Settings
  3. Navigate to Security tab
  4. Find API Key section and copy your token
  5. Paste it in the form below and click Save

About KoboToolbox

What is KoboToolbox?

KoboToolbox is a free, open-source platform for mobile data collection. It allows field staff to collect data offline using mobile devices.

Features:

  • Offline data collection
  • GPS location capture
  • Photo/audio/video capture
  • Automatic sync to UPG MIS
  • Works on Android devices
